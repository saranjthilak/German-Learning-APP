# ⚡ Quick Start Guide - German Vocab Game

## 🚀 30-Second Setup

### 1️⃣ Install Node.js (if needed)
Download from: https://nodejs.org (LTS version)
Choose default options, restart computer

### 2️⃣ Open PowerShell in Project Folder
```
c:\Users\SARAN J THILAK\New projects\German app
```
(Right-click folder → "Open PowerShell window here")

### 3️⃣ Install Dependencies
```powershell
npm install
```
(Wait 2-5 minutes)

### 4️⃣ Start the Game
```powershell
npm run dev
```

### 5️⃣ Open Browser
- Auto-opens at http://localhost:3000
- Or manually open in browser

---

## 🎮 First Time Playing

1. **Click ⚙️ Settings** → Set your player name
2. **Choose a game** (5 types available)
3. **Select difficulty** (10, 15, or 20 words)
4. **Play and earn XP** 🎯
5. **Check Dashboard** for progress

## 📚 5 Game Types

| Game | How to Play | Best For |
|------|------------|----------|
| 🎯 **Matching** | Drag German ↔ English pairs | Visual learners |
| 🧠 **Memory** | Flip cards to find pairs | Pattern recognition |
| ❓ **Quiz** | Choose correct English translation | Quick learning |
| ⌨️ **Typing** | Type German word for English | Spelling practice |
| 🔊 **Pronunciation** | Speak into microphone | Oral practice |

## ⚠️ Troubleshooting

**Problem**: npm not found
- ✅ Restart computer after installing Node.js
- ✅ Or reinstall Node.js

**Problem**: Port 3000 in use
- ✅ Run: `npm run dev -- --port 3001`

**Problem**: "Cannot find module"
- ✅ Run: `npm install` again

**Problem**: Speech not working
- ✅ Use Chrome or Edge browser
- ✅ Check microphone is enabled
- ✅ Grant microphone permission
